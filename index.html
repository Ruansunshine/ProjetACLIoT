ACLIoT – Climatização e Iluminação com IoT

Explorando Soluções Inteligentes para Eficiência Energética Atualmente em fase de desenvolvimento, o projeto ACLIoT integra tecnologias de IoT para monitoramento básico de climatização e iluminação. O sistema inicial foi projetado para coletar dados de temperatura, luminosidade e consumo de energia, preparando o terreno para uma automação mais avançada e inteligente no futuro. O projeto realiza a coleta de dados por meio de sensores de temperatura, luminosidade e consumo de energia, com uma estrutura preparada para expansão e controle automatizado. O ACLIoT está evoluindo para se tornar uma solução completa de automação, com foco em otimizar o uso de energia e melhorar o conforto dos usuários.
